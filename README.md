# სახლის ფასების პროგნოზირება - Kaggle Competition

## Kaggle-ის კონკურსის მოკლე მიმოხილვა

ეს პროექტი მონაწილეობს Kaggle-ის "House Prices: Advanced Regression Techniques" კონკურსში. მიზანია საცხოვრებელი სახლების გაყიდვის ფასების პროგნოზირება 79 ფიჩერზე დაყრდნობით. შეფასების მეტრიკაა RMSLE (Root Mean Squared Logarithmic Error).

## მიდგომა პრობლემის გადასაჭრელად

მოდელის შექმნის პროცესი მოიცავდა რამდენიმე ძირითად ეტაპს:

2. Feature Engineering

5. N/A მნიშვნელობების შევსება (სვეტების ტიპების მიხედვით)

6. ფიჩერების საბოლოო სელექცია RFE-ით (Recursive Feature Elimination)
გამოიყენე RFE Random Forest ესთიმატორით, რომ შვეერჩია X ყველაზე მნიშვნელოვანი ფიჩერი.

// xgboost-0.138. x=30 -> 0.14, x=50 -> 0.137, x=60 -> 0.135.
// ვალიდაციაზე 40,50,60 თითქმის იგივე პასუხს მაძლევდა უბრალოდ ტესტზე 40-მა აჯობა (0.140, 0.142, 0.141).


RFE მუშაობს შემდეგნაირად: რეკურსიულად შლის ყველაზე ნაკლებად მნიშვნელოვან ფიჩერებს, სანამ არ მიაღწევს სასურველ რაოდენობას (40).

ამ ნაბიჯმა კიდევ უფრო შეამცირა განზომილება, გაამარტივა მოდელი და გააუმჯობესა განზოგადების უნარი.


7. მოდელის ტრეინინგი და ექსპერიმენტები
Random Forest ბაზური მოდელი:  Random Forest 15 ესთიმატორით, როგორც საწყისი ბაზური მოდელი.

XGBoost საბოლოო მოდელი: მოამზადე XGBoost მოდელი, როგორც საბოლოო პროგნოზირების მოდელი, შემდეგი ჰიპერპარამეტრებით:

n_estimators=1000 (დიდი რაოდენობის ხეები სტაბილური სწავლისთვის)

max_depth=5 (აკონტროლებს ხის სირთულეს ოვერფიტინგის თავიდან ასაცილებლად)

learning_rate=0.04 (მცირე ნაბიჯი ნელი, სტაბილური სწავლისთვის)

შევაფასე მოდელი ვალიდაციის მონაცემებზე საბოლოო RMSE-ს მისაღებად.

ასევე ვცადე linear regression თუმცა ძალიან პერფორმანსი ჰქონდა ტავიდანვე და ზედა 2-შ ავარჩიე.

მოდელების საუკეთეო შეგედები(rmse), გალოგარითმების გარეშე აცდენები: lin. regression -> 35000, randomForest -> 27707, xgBoost -> 23801(best, log -> 0.138).
